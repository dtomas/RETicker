I. Installation

Das Programm benötigt eine Version der ROX-Lib2 größer 1.9.13. Zum Installieren einfach das Archiv entpacken und los gehts. (Ein Python-Interpreter sollte auf eurem Rechner installiert sein)
Signatur prüfen:
#>gpg Ticker_0.4.5.tgz.sig
Programm entpacken:
#>tar xfz Ticker_0.4.5.tgz

Programm-Start:

Es gibt verschiedene Möglichkeiten das Programm zu starten:
  1. Im ROX-Filer auf das Symbol Ticker_0.4.5 klicken
  2. Das Symbol Ticker_0.4.5 aus dem ROX-Filer auf das Panel ziehen
  3. Das Programm in einer shell starten.
    #>./start.py --help
    *** RETicker v 0.4.5 ***
    usage: ./start.py [--config X][--verbose|-v][--help]
    --config X start with Konfiguration X
    use any Name for X
    --verbose be more verbose
    --help shows this help ;)
    Please report any Problems (see README.txt)
